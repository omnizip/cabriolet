=== Analysis

**Speed Winner**: MSZIP offers the best balance of compression speed (57 MB/s) and ratio (70%).

**Ratio Winner**: LZX (2MB window) achieves 85.5% average compression but at 9.6 MB/s.

**Memory Winner**: LZSS uses minimal memory (22MB compression) with decent 57% ratio.
