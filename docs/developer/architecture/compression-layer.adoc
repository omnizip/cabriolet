== Compression Layer

=== Decompressor Architecture

Base class with algorithm-specific implementations:

* **Base**: Common decompression interface
* **MSZIP**: DEFLATE implementation
* **LZX**: Advanced dictionary compression
* **Quantum**: Arithmetic coding
* **LZSS**: Simple dictionary compression

=== Design Pattern

Strategy pattern for compression algorithms.
