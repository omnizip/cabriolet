== Format Layer

=== Format Support

Each format has:

* **Parser**: Reads file structure
* **Decompressor**: Extracts content
* **Compressor**: Creates archives
* **Models**: Data structures

Formats: CAB, CHM, SZDD, KWAJ, HLP, LIT, OAB

=== Design

Each format is self-contained with minimal dependencies.
