== Troubleshooting

=== Common Problems

==== File Not Found

**Problem**: Cannot open file

**Solutions**:
* Check file path
* Verify file permissions
* Ensure file exists

==== Checksum Errors

**Problem**: Checksum verification failed

**Solutions**:
* Re-download file
* Use salvage mode
* Disable checksum verification (risky)

==== Unsupported Format

**Problem**: Format not recognized

**Solutions**:
* Verify file format
* Check file signature
* Update Cabriolet to latest version

==== Out of Memory

**Problem**: Memory exhausted

**Solutions**:
* Use streaming mode
* Reduce buffer sizes
* Process files individually

==== Extraction Fails

**Problem**: Cannot extract files

**Solutions**:
* Check disk space
* Verify write permissions
* Try salvage mode
* Check for corrupted archive
