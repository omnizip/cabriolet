== Guides

=== Purpose

This section provides comprehensive guides for using Cabriolet effectively. From basic operations to advanced techniques, these guides cover all aspects of working with Microsoft compression formats.

=== References

* <<../getting-started/index#,Getting Started>>
* <<../reference/index#,Reference Documentation>>

=== Concepts

Cabriolet supports multiple compression formats and algorithms, each with specific use cases. These guides help you understand when and how to use each format, choose appropriate compression algorithms, and handle advanced scenarios.

=== Guide Categories

==== Basic Usage

<<basic-usage/index#,Basic Usage Guides>> cover fundamental operations:

* Extracting files from archives
* Creating new archives
* Listing archive contents
* Testing archive integrity

==== Format Guides

<<formats/index#,Format-Specific Guides>> document each of the 7 supported formats:

* <<formats/cab-format#,CAB>> - Microsoft Cabinet files
* <<formats/chm-format#,CHM>> - Compiled HTML Help
* <<formats/szdd-format#,SZDD>> - Single-file LZSS compression
* <<formats/kwaj-format#,KWAJ>> - KWAJ compression format
* <<formats/hlp-format#,HLP>> - Windows Help files
* <<formats/lit-format#,LIT>> - Microsoft Reader eBooks
* <<formats/oab-format#,OAB>> - Offline Address Book

==== Compression Algorithms

<<compression/index#,Compression Guides>> explain the 5 supported algorithms:

* <<compression/choosing-algorithm#,Choosing an Algorithm>> - Selection guide
* <<compression/none-compression#,None>> - Uncompressed storage
* <<compression/lzss-compression#,LZSS>> - Dictionary compression
* <<compression/mszip-compression#,MSZIP>> - ZIP/Deflate compatible
* <<compression/lzx-compression#,LZX>> - High compression ratio
* <<compression/quantum-compression#,Quantum>> - Memory-efficient

==== Advanced Usage

<<advanced-usage/index#,Advanced Guides>> cover specialized topics:

* <<advanced-usage/multi-part-cabinets#,Multi-part Cabinets>> - Split archives
* <<advanced-usage/embedded-cabinets#,Embedded Cabinets>> - Extracting from executables
* <<advanced-usage/custom-io-handlers#,Custom I/O Handlers>> - Memory and streaming
* <<advanced-usage/error-handling#,Error Handling>> - Robust error management
* <<advanced-usage/salvage-mode#,Salvage Mode>> - Recovering damaged archives
* <<advanced-usage/performance-tuning#,Performance Tuning>> - Optimization techniques
* <<advanced-usage/memory-operations#,Memory Operations>> - In-memory processing

=== Learning Path

Recommended progression through the guides:

1. **Start with Basics**: Master <<basic-usage/index#,basic operations>>
2. **Learn Your Format**: Study the <<formats/index#,format guide>> for your use case
3. **Understand Compression**: Review <<compression/index#,compression algorithms>>
4. **Advanced Techniques**: Explore <<advanced-usage/index#,advanced topics>> as needed
