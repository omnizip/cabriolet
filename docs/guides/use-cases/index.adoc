== Use Cases

This section provides practical guides for common Cabriolet use cases, demonstrating how to apply the library to real-world scenarios.

=== Common Patterns

==== Pattern 1: Extract-Transform-Archive

1. Extract from legacy format
2. Transform/process content
3. Re-archive in modern format

**Use Cases**: Legacy conversion, format migration

==== Pattern 2: Distribute-Verify-Install

1. Create distribution archive
2. Verify integrity
3. Deploy/install on target

**Use Cases**: Software distribution, deployment

==== Pattern 3: Scan-Extract-Process

1. Scan for archives
2. Extract relevant files
3. Process/analyze content

**Use Cases**: Documentation extraction, data mining

=== Integration Examples

==== Ruby on Rails Application

[source,ruby]
----
# In your Rails app
class ArchiveController < ApplicationController
  def extract
    archive = params[:file]
    cab = Cabriolet::CAB::Parser.new.parse(archive.tempfile.path)

    cab.files.each do |file|
      # Process extracted files
      process_file(file.name, file.data)
    end
  end
end
----

==== Rake Task

[source,ruby]
----
# lib/tasks/archive.rake
namespace :archive do
  desc "Extract and process CAB files"
  task :process => :environment do
    Dir.glob('imports/*.cab').each do |cab_file|
      Cabriolet::CAB::Parser.new.parse(cab_file).files.each do |file|
        # Process file
      end
    end
  end
end
----

==== Command Line Script

[source,ruby]
----
#!/usr/bin/env ruby
require 'cabriolet'

ARGV.each do |file|
  cab = Cabriolet::CAB::Parser.new.parse(file)
  puts "#{file}: #{cab.files.count} files"
end
----
